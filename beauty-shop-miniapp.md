# 美容院微信小程序完整方案

## 一、核心目标

**让美容院更挣钱**：
1. **拉新** - 让陌生人变成客户
2. **留存** - 让客户多次消费
3. **裂变** - 让客户带新客户

## 二、小程序功能架构

### 1. 客户端（面向消费者）

#### 🎯 核心功能

**首页（获客入口）**
```
├─ 新人礼包（9.9 元体验卡，限首次）
├─ 热门项目推荐（带"距你 1.2km"）
├─ 团购套餐（3 人拼团 8 折）
├─ 会员专享价（突出"省 ¥200"）
└─ 客户评价（带真实照片）
```

**预约系统**
```
├─ 实时预约（显示美容师空闲时间）
├─ 智能推荐（根据上次服务推荐）
├─ 一键取消（提前 24 小时免费）
└─ 预约提醒（微信模板消息）
```

**会员中心（留存核心）**
```
├─ 积分商城（消费 1 元=1 积分，可换礼品）
├─ 会员等级（银卡/金卡/钻石卡，享不同折扣）
├─ 专属客服（专属微信）
├─ 生日礼包（生日月双倍积分）
└─ 老带新奖励（推荐 1 人得¥100 券）
```

**营销活动（裂变工具）**
```
├─ 拼团活动（3 人成团，享 7 折）
├─ 砍价活动（邀请好友助力，最高 5 折）
├─ 秒杀活动（每周三 10 点，9.9 元体验）
├─ 分享有礼（分享海报，好友消费得佣金）
└─ 打卡活动（连续到店 10 次送项目）
```

#### 📊 数据分析（让客户看到价值）
```
├─ 消费记录（历史订单）
├─ 剩余次数/余额
├─ 项目效果对比（前后照片）
└─ 专属顾问（美容师微信）
```

## 三、获客与留存策略

### 🎯 拉新策略（0→1）

**1. 低价引流**
```
├─ 9.9 元体验卡（原价¥299）
├─ 限首次客户
├─ 需到店核销
└─ 核销时转化正价会员
```

**2. 拼团裂变**
```
├─ 3 人成团享 7 折
├─ 每人限 1 张
├─ 7 天有效期
└─ 未成团自动退款
```

**3. 老带新奖励**
```
├─ 老客分享专属码
├─ 新客用码下单，老客得¥100
├─ 新客也得¥50 优惠券
└─ 双向奖励，双方受益
```

**4. 地理位置推广**
```
├─ 显示"距你 1.2km"
├─ 附近 3km 用户优先推送
├─ 结合微信附近的小程序
└─ 投放朋友圈广告（定向周边）
```

### 🔄 留存策略（1→N）

**1. 会员等级体系**
```
银卡（消费¥1000+）：95 折
金卡（消费¥3000+）：9 折 + 生日礼包
钻石卡（消费¥10000+）：85 折 + 专属客服 + 优先预约
```

**2. 积分系统**
```
消费 1 元=1 积分
积分可换：
├─ 兑换项目（¥100=1000 积分）
├─ 兑换礼品（面膜/护肤品）
├─ 抵扣现金（100 积分=1 元）
└─ 兑换特权（免费升级房型）
```

**3. 定期活动**
```
├─ 每月 8 号会员日（双倍积分）
├─ 每周三秒杀（9.9 元体验）
├─ 季度抽奖（消费满¥500 抽奖）
└─ 节日礼包（三八节/母亲节/七夕）
```

**4. 个性化服务**
```
├─ 根据皮肤类型推荐项目
├─ 根据上次服务推荐后续项目
├─ 根据消费习惯推荐套餐
└─ 生日/纪念日专属优惠
```

**5. 提醒机制**
```
├─ 预约提醒（提前 24 小时）
├─ 项目到期提醒（剩余 1-2 次）
├─ 优惠券到期提醒（提前 3 天）
└─ 生日提醒（提前 7 天发礼包）
```

## 四、技术实现

### 前端（微信小程序）

**技术栈**
```javascript
{
  框架：原生微信小程序
  UI 组件：Vant Weapp / TDesign
  状态管理：MobX-miniprogram
  请求库：Taro Request / 小程序原生
  地图：腾讯地图 API
}
```

**核心页面**
```
├─ pages/index（首页）
├─ pages/booking（预约）
├─ pages/member（会员中心）
├─ pages/activity（营销活动）
├─ pages/order（订单）
├─ pages/user（用户中心）
└─ pages/admin（员工端）
```

### 后端

**技术栈**
```javascript
{
  框架：Node.js + Express / Koa
  数据库：MySQL / PostgreSQL
  ORM：Sequelize / TypeORM
  缓存：Redis
  文件存储：腾讯云 COS
  消息推送：微信模板消息
}
```

**数据库设计**
```sql
-- 用户表
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  openid VARCHAR(100) UNIQUE,
  phone VARCHAR(20),
  nickname VARCHAR(50),
  avatar TEXT,
  gender BOOLEAN,
  birthday DATE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 会员表
CREATE TABLE members (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  level VARCHAR(20), -- silver/gold/diamond
  points INTEGER DEFAULT 0,
  balance DECIMAL(10,2) DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 项目表
CREATE TABLE projects (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100),
  description TEXT,
  price DECIMAL(10,2),
  duration INTEGER, -- 时长（分钟）
  image TEXT,
  is_hot BOOLEAN DEFAULT false
);

-- 预约表
CREATE TABLE bookings (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  project_id INTEGER REFERENCES projects(id),
  staff_id INTEGER, -- 美容师 ID
  appointment_time TIMESTAMP,
  status VARCHAR(20), -- pending/completed/cancelled
  created_at TIMESTAMP DEFAULT NOW()
);

-- 订单表
CREATE TABLE orders (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  amount DECIMAL(10,2),
  payment_method VARCHAR(20),
  status VARCHAR(20), -- unpaid/paid/refunded
  created_at TIMESTAMP DEFAULT NOW()
);

-- 优惠券表
CREATE TABLE coupons (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100),
  discount DECIMAL(5,2), -- 折扣
  min_amount DECIMAL(10,2), -- 最低消费
  valid_days INTEGER, -- 有效期（天）
  total_count INTEGER,
  used_count INTEGER DEFAULT 0,
  is_active BOOLEAN DEFAULT true
);
```
